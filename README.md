# FPCH - Function of Chaotic Hyperbolic Permutation

**An Invitation to Cryptanalysis**

FPCH is a cryptographic hash function exploring chaotic dynamics in real quadratic fields using exact integer arithmetic.

## 📄 Paper
- [FPCH Paper (PDF)](FPCH_Paper.pdf)
- [LaTeX Source](source.tex)

## 🎯 Purpose
This repository contains a preliminary cryptographic construction intended as an **invitation to cryptanalysis**. We make no claims of provable security; instead, we explicitly solicit attacks from the cryptographic community.
